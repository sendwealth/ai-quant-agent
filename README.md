## 📈 推荐策略

### 1. 保守策略 ⭐⭐⭐⭐⭐

**适用场景**: 下跌/震荡市场

```python
# 配置
MA: 5/30
ATR止损: 2.5倍
追踪止损: 2.0倍
最大仓位: 25%
```

**表现**: 五粮液+2.82% (超额+19.4%)

### 2. 波动率自适应v11 ⭐⭐⭐⭐

**适用场景**: 上涨市场

```python
# 配置
高波动(>35%): 仓位15%, 止损2.0x
正常波动: 仓位25%, 止损2.5x
低波动(<15%): 仓位35%, 止损3.0x
```

**表现**: 比亚迪+9.46% (夏普0.88)

### 3. 专业级v15 ⭐⭐⭐⭐

**适用场景**: 全市场

```python
# 配置
风险规则: 1% (单笔交易)
交易成本: 0.03%
滑点: 0.1%
样本外测试: 70/30分割
```

**表现**: Profit Factor 5.23
