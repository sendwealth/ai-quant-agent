**目标**: 从信号生成到自动下单

**实现**:
```python
# auto_trader.py
class AutoTrader:
    def __init__(self):
        self.broker = Broker()  # 券商API
        self.risk = RiskManager()
        
    def execute_signal(self, signal):
        # 1. 风险检查
        if not self.risk.check(signal):
            return
        
        # 2. 计算仓位
        position = self.calculate_position(signal)
        
        # 3. 执行交易
        order = self.broker.place_order(
            symbol=signal.symbol,
            side=signal.action,
            quantity=position.shares
        )
        
        # 4. 记录日志
        self.log_trade(order)
```

### 3.1 集中持仓策略（学习InStreet）
**核心理念**: 选最强股，全仓操作

**实现**:
```python
class ConcentratedStrategy:
    def select_top_stock(self, candidates):
        """选择最强的1只股票"""
        scores = []
        for stock in candidates:
            score = (
                stock.momentum * 0.4 +      # 动量
                stock.volume_ratio * 0.3 +   # 量比
                stock.trend_strength * 0.3   # 趋势强度
            )
            scores.append((stock, score))
        
        return sorted(scores, key=lambda x: x[1], reverse=True)[0][0]
    
    def calculate_position(self, stock):
        """全仓操作（保留5%现金）"""
        return self.capital * 0.95 / stock.price
```

## ⚠️ 风险控制

### 止损机制
1. **单笔止损**: -6%自动卖出
2. **日止损**: 当日亏损-3%停止交易
3. **周止损**: 周亏损-5%暂停策略
4. **月止损**: 月亏损-10%全面复盘

### 仓位控制
- 单只股票: ≤30%
- 总仓位: ≤80%
- 现金比例: ≥20%

### 熔断机制
- 连续3天亏损 → 降低仓位50%
- 连续5天亏损 → 暂停交易
- 单日亏损-5% → 立即清仓
