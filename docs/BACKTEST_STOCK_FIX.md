# 🔍 回测股票选择问题修复

**问题**: 每次回测都是旧的4只股票（300750, 002475, 601318, 600276）

**原因**:
1. scripts/full_backtest.py 硬编码了旧股票列表（作为fallback）
2. 动态选股结果未同步到回测系统
3. 新选出的股票（000895, 000333, 000538）没有生成信号数据
