## 🚀 立即开始

### 方式1：手动运行（每天5秒）

```bash
cd ai-quant-agent
python3 examples/auto_trading_bot.py
```

**运行时间**: 每天收盘后（15:00后）
**耗时**: 5秒
**输出**: 显示信号、交易、持仓

### 方式2：自动运行（推荐）

#### macOS/Linux

创建每日定时任务：

```bash
# 编辑crontab
crontab -e

# 添加以下行（每天15:30自动运行）
30 15 * * 1-5 cd /Users/rowan/clawd/projects/ai-quant-agent && /usr/bin/python3 examples/auto_trading_bot.py >> logs/auto_trading.log 2>&1
```

#### Windows

使用任务计划程序：
1. 打开"任务计划程序"
2. 创建基本任务
3. 触发器：每天 15:30
4. 操作：启动程序
   - 程序：`python.exe`
   - 参数：`examples/auto_trading_bot.py`
   - 起始位置：`C:\path\to\ai-quant-agent`

## 🎯 交易规则

### 买入条件（全部满足）
- ⚡ MA快线上穿慢线（金叉）
- ✅ MACD柱状图为正
- ✅ RSI在30-70之间
- ✅ 初始仓位30%

### 卖出条件（满足任一）
- 🔴 止损: -8%
- 💰 止盈1: +10% (卖50%)
- 💰 止盈2: +20% (清仓)
- 🔄 趋势反转: MA死叉 + MACD负

### 仓位管理
- 初始: 30%
- 最大: 50%
- 止损: -8%
- 移动止损: 跟随最高价
