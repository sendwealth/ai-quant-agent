## 🎯 策略逻辑

### 买入信号

```
IF (MA快线 > MA慢线) 
   AND (MACD > 信号线)
   AND (30 < RSI < 70)
THEN 买入
```

**解释**:
1. **MA金叉**: 趋势向上
2. **MACD确认**: 动量增强
3. **RSI过滤**: 避免追高

### 卖出信号

```
IF (MA快线 < MA慢线) 
   OR (RSI > 80)
THEN 卖出
```

**解释**:
1. **MA死叉**: 趋势反转
2. **RSI超买**: 过热风险

### 风险控制

```
IF (亏损 >= 8%) THEN 止损卖出
IF (盈利 >= 10%) THEN 卖出50%
IF (盈利 >= 20%) THEN 清仓
```

## 🔧 技术指标

### MA均线

**计算**:
```python
MA = close.rolling(period).mean()
```

**参数**:
- 快线: 8-10
- 慢线: 25-35

**信号**:
- 金叉: 快线上穿慢线 (买入)
- 死叉: 快线下穿慢线 (卖出)

### MACD

**计算**:
```python
MACD = EMA(close, 12) - EMA(close, 26)
Signal = EMA(MACD, 9)
Histogram = MACD - Signal
```

**信号**:
- MACD > Signal: 多头 (确认买入)
- MACD < Signal: 空头 (确认卖出)

### RSI

**计算**:
```python
RSI = 100 - 100 / (1 + RS)
RS = 平均涨幅 / 平均跌幅
```

**参数**:
- 周期: 14

**信号**:
- RSI < 30: 超卖 (可买入)
- RSI > 70: 超买 (可卖出)

### ATR

**计算**:
```python
TR = max(H-L, |H-C_prev|, |L-C_prev|)
ATR = TR.rolling(14).mean()
```

**用途**:
- 动态止损: ATR × 倍数
- 仓位管理: 波动率调整

## 💰 仓位管理

### 固定权重

```python
仓位 = 总资金 × 股票权重
股数 = 仓位 / 当前价格
```

**示例**:
- 总资金: 100,000
- 宁德时代权重: 45%
- 价格: 350
- 买入股数: 100,000 × 0.45 / 350 = 128股

### 动态调整 (可选)

```python
IF 牛市:
    仓位 = 1.2 × 基础仓位
IF 熊市:
    仓位 = 0.5 × 基础仓位
IF 震荡:
    仓位 = 0.8 × 基础仓位
```
