创建一个Excel文件，包含以下列：

| A列 | B列 | C列 | D列 | E列 | F列 | G列 | H列 | I列 | J列 | K列 | L列 | M列 | N列 | O列 |
|-----|-----|-----|-----|-----|-----|-----|-----|-----|-----|-----|-----|-----|-----|-----|
| 日期 | 股票代码 | 股票名称 | 操作 | 数量 | 价格 | 金额 | 手续费 | 现价 | 盈亏 | 盈亏% | 信号 | 止损价 | 止盈价 | 备注 |

## 📊 统计公式

### Excel公式

**总资产**:
```
=现金+SUMPRODUCT(持仓数量,现价)
```

**总收益**:
```
=总资产-初始资金
```

**收益率**:
```
=(总资产-初始资金)/初始资金
```

**单只股票盈亏**:
```
=(卖出价-买入价)*数量-手续费
```

**胜率**:
```
=盈利次数/总交易次数
```

**夏普比率** (简化版):
```
=AVERAGE(日收益率)/STDEV(日收益率)*SQRT(252)
```
