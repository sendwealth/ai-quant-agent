## ✅ 买入条件（必须全部满足）

```
□ MA快线上穿慢线（金叉）
□ MACD柱状图为正（>0）
□ RSI在30-70之间
□ 成交量放大（>20日均量）
```

**买入操作**:
1. 确认4个条件全部满足 ✅
2. 计算买入金额（初始30%）
3. 下单买入
4. 设置止损价（成本-8%）
5. 设置止盈1（成本+10%）
6. 设置止盈2（成本+20%）
7. 记录交易日志

## ⚠️ 卖出条件（满足任一）

### 1. 止损 🔴
```
价格 ≤ 成本价 × 0.92
→ 立即全部卖出
```

### 2. 止盈1 💰
```
价格 ≥ 成本价 × 1.10
→ 卖出50%仓位
```

### 3. 止盈2 💰
```
价格 ≥ 成本价 × 1.20
→ 全部清仓
```

### 4. 趋势反转 🔄
```
MA快线下穿慢线 且 MACD<0
→ 全部卖出
```

**卖出操作**:
1. 确认卖出条件 ✅
2. 计算卖出数量
3. 下单卖出
4. 记录盈亏
5. 更新交易日志

## 📞 紧急情况

### 如果亏损达到-8%
```
1. 立即止损，不要犹豫
2. 全部卖出该股票
3. 记录亏损
4. 分析原因
5. 等待下次机会
```

### 如果组合亏损达到-12%
```
1. 减仓至50%
2. 重新评估策略
3. 检查是否参数失效
4. 考虑暂停交易
```

```
1. 暂停交易1周
2. 复盘分析原因
3. 检查市场环境
4. 调整心态
```
