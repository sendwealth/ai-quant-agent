## 🎯 项目目标

打造一个基于多智能体协同的AI原生量化交易系统，通过以下方式实现盈利：
1. **策略智能体** - 从自然语言自动生成交易策略代码
2. **分析智能体** - 实时市场分析、趋势识别、情绪分析
3. **风控智能体** - 动态风险管理、止损止盈、仓位管理
4. **优化智能体** - 强化学习参数优化，每天测试5000+组合
5. **多智能体对抗** - 20个AI策略互博，找出最优组合

### ✅ 核心智能体开发

#### 1. 策略智能体 (StrategyAgent)
**文件：** `agents/strategy_agent.py`

**功能：**
- 从自然语言描述自动生成可执行Python代码
- 支持30秒策略生成
- 集成LLM（OpenAI/智谱AI）
- 自动解析策略参数、开仓/平仓条件
- 策略性能分析（收益率、夏普比率、最大回撤等）

**示例策略：**
```python
# 自然语言描述：
"当20日均线向上突破60日均线时买入，当20日均线向下突破60日均线时卖出"

# 自动生成完整的Python策略代码，包括：
# - generate_signals函数
# - 技术指标计算
# - 信号生成逻辑
```

#### 2. 分析智能体 (AnalysisAgent)
**文件：** `agents/analysis_agent.py`

**功能：**
- 趋势分析（SMA、ADX）
- 动量分析（RSI、MACD）
- 波动率分析（ATR、布林带）
- 成交量分析
- 支撑/阻力位识别
- 综合市场情绪判断
- 交易信号生成

**输出示例：**
```
【趋势分析】
方向: strong_uptrend
强度: 强
ADX: 42.5

【动量分析】
状态: bullish
RSI: 55.3
MACD: bullish

【综合情绪】
🎯 VERY_BULLISH
```

#### 3. 风控智能体 (RiskAgent)
**文件：** `agents/risk_agent.py`

**功能：**
- 智能仓位计算（基于信号强度和波动率）
- 动态止损（ATR止损、波动率止损）
- 盈利加仓（追踪止损）
- 风险限制检查（日亏损限制、最大回撤限制）
- 持仓集中度管理
- VaR计算
- 投资组合风险评估

**核心特性：**
- 高波动率自动降低仓位
- 盈利时移动止损保护利润
- 多策略组合风险分散
