## 📅 操作流程

### 第1步: 开户（如未开户）

**时间**: 1-3个工作日

**流程**:
```
1. 选择券商（推荐: 华泰、中信、招商）
2. 线上开户（需身份证+银行卡）
3. 等待审核（1-2天）
4. 开通交易权限
5. 银证转账
```

**融资融券开通**（如需杠杆）:
```
条件: 50万资产 + 半年交易经验
流程: 线上申请 → 风险测评 → 签署协议
时间: 3-5个工作日
```

---

### 第2步: 资金转入

**时间**: 实时或T+1

**操作**:
```
1. 登录银行APP
2. 选择"银证转账"
3. 输入金额（¥10,000 - 170,000）
4. 确认转账
5. 查看证券账户余额
```

**注意事项**:
- ⚠️ 转账时间: 交易日 9:00-16:00
- ⚠️ 当天转入，当天可用
- ⚠️ 确认资金到账后再操作

---

### 第3步: 建仓（买入）

**时间**: 1天（建议上午）

**方案A: 无杠杆建仓**
```
总资金: ¥100,000

配置:
  宁德时代: ¥70,000 (70%)
    买入价: ¥390-400
    买入量: ~175股
  
  立讯精密: ¥20,000 (20%)
    买入价: ¥45-48
    买入量: ~430股
  
  现金: ¥10,000 (10%)
    用途: 应急/补仓
```

**方案B: 1.5倍杠杆建仓**
```
本金: ¥100,000
融资: ¥50,000
总额: ¥150,000

配置:
  宁德时代: ¥150,000 (100%)
    买入价: ¥390-400
    买入量: ~375股
```

**操作步骤**:
```
1. 9:30 开盘
2. 观察开盘价（不追高）
3. 分2-3批买入
4. 记录买入价格和数量
5. 设置止损单（如需）
```

**建仓技巧**:
```
✅ 不要一次性买入（分批建仓）
✅ 避开开盘前30分钟（波动大）
✅ 观察成交量（量价配合）
✅ 设置限价单（控制成本）
```

---

### 第4步: 持有（1-2年）

**每周检查**（5分钟）:
```
□ 登录证券APP
□ 查看持仓市值
□ 查看收益率
□ 检查是否触及止损线
□ 不操作，只观察
```

**每月复盘**（30分钟）:
```
□ 记录月度收益
□ 对比目标进度
□ 评估市场环境
□ 决定是否调整
```

**触发条件**:
```
止损（方案B）:
  - 净值跌破¥75,000（-25%）
  - 立即平仓，不犹豫

止盈:
  - 持有满1年
  - 年化收益达到50%+
  - 考虑部分止盈或降杠杆
```

---

### 第5步: 退出（1-2年后）

**退出条件**:
```
✅ 达到目标收益（年化30%+）
✅ 持有满1-2年
✅ 需要用钱
✅ 基本面恶化
✅ 触发止损
```

**退出步骤**:
```
1. 检查当前收益
2. 计算税费（印花税0.1%）
3. 分批卖出（避免冲击成本）
4. 偿还融资（如有）
5. 提取资金
6. 记录盈亏
7. 评估是否继续下一轮
```

## ⚠️ 风险管理

### 核心原则

1. **严格止损**（如有杠杆）
   ```
   止损线: 净值-25%
   触发后: 立即平仓，不犹豫
   原因: 保护本金，避免爆仓
   ```

2. **不追涨杀跌**
   ```
   ✅ 分批建仓
   ✅ 设置限价单
   ❌ 不追高买入
   ❌ 不恐慌卖出
   ```

3. **长期持有**
   ```
   持有时间: 1-2年
   检查频率: 每周5分钟
   操作频率: 尽量少操作
   ```

4. **情绪控制**
   ```
   ✅ 接受短期波动（±10%）
   ✅ 关注长期趋势
   ✅ 相信基本面
   ❌ 不受短期噪音影响
   ```

---

### 应急预案

**情景1: 短期回调10%**
```
现象: 股价下跌10%
操作: 持有，不操作
理由: 正常波动
```

**情景2: 中期回调20%**
```
现象: 股价下跌20%
操作: 评估是否补仓
理由: 可能是加仓机会
条件: 有充足现金 + 相信长期价值
```

**情景3: 大幅回调30%**
```
现象: 股价下跌30%
操作（无杠杆）: 持有或加仓
操作（有杠杆）: ⚠️ 接近止损线
  - 方案A: 补仓¥20,000
  - 方案B: 减仓降风险
  - 方案C: 止损退出
```

**情景4: 基本面恶化**
```
现象: 公司业绩下滑/行业利空
操作: 立即平仓
理由: 避免更大损失
```

## 💰 费用说明

### 交易成本

**买入**:
```
佣金: 0.025%（最低5元）
过户费: 0.001%
合计: 约0.026%
```

**卖出**:
```
佣金: 0.025%（最低5元）
印花税: 0.1%
过户费: 0.001%
合计: 约0.126%
```

**往返成本**: 约0.152%

**示例**:
```
买入¥10万:
  - 佣金: ¥25
  - 过户费: ¥1
  - 合计: ¥26

卖出¥10万:
  - 佣金: ¥25
  - 印花税: ¥100
  - 过户费: ¥1
  - 合计: ¥126

往返成本: ¥152（0.152%）
```

---

### 融资成本（如需）

**融资利率**:
```
年化利率: 6%
日利率: 0.0164%

示例:
借款¥5万 × 6% = ¥3,000/年
借款¥5万 × 0.0164% = ¥8.22/天
```

**注意事项**:
- ⚠️ 利息按日计算
- ⚠️ 可随时还款
- ⚠️ 持有时间越长，成本越高
