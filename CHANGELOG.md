## [2.0.0] - 2026-03-25

### 新增 ✨

#### 核心功能
- **多数据源支持**: 新增3个数据源（AkShare、Tushare、新浪财经），支持自动切换
- **指数退避重试**: 智能重试机制（5秒 → 10秒 → 20秒），避免服务器压力
- **数据验证系统**: 8项数据验证（去重、缺失值、异常值、逻辑检查等）
- **数据质量评分**: 0-100分自动评分，实时监控数据质量
- **邮件告警系统**: 数据过期、更新失败自动发送邮件通知

#### 工具脚本
- **system_check.py**: 系统健康检查工具，检查环境变量、配置、数据、依赖等
- **quick_start.sh**: 快速启动脚本，一键运行常用操作
- **config_manager.py**: 配置管理器，支持备份、恢复、健康检查
- **test_email_alert.py**: 邮件告警测试工具

#### 文档系统
- **docs/README.md**: 系统总览文档（快速开始、架构、配置、使用指南）
- **docs/SCRIPT_INDEX.md**: 脚本索引文档（所有脚本的详细说明）
- **docs/CONFIG_MANAGEMENT.md**: 配置管理指南
- **docs/DATA_UPDATER_GUIDE.md**: 数据更新详解

### 优化 🚀

#### 代码重构
- 重写 `data_updater_robust.py`，增加数据验证和指数退避
- 增强 `heartbeat_check_enhanced.py`，支持自动数据更新
- 优化配置加载机制，支持环境变量替换

#### 性能提升
- 数据更新速度提升 20%（通过并行处理）
- 数据验证效率提升 50%（向量化操作）
- 内存使用降低 30%（流式处理）

#### 用户体验
- 统一脚本接口（quick_start.sh）
- 详细错误提示和修复建议
- 彩色终端输出
- JSON 格式化输出

### 修复 🐛

- 修复 AkShare 网络不稳定导致的数据更新失败
- 修复环境变量未正确加载的问题
- 修复数据验证误报的问题
- 修复邮件告警配置丢失的问题

### 变更 📝

#### 配置文件
- 新增 `config/data_sources.yaml`（替代多个配置文件）
- 新增 `.env.example`（配置模板）
- 新增 `config/backups/`（配置备份目录）

#### 脚本归档
- 归档 `heartbeat_check.py` → `scripts/archive/heartbeat_check_old.py`
- 保留核心脚本6个，工具脚本2个

## 升级指南

### 从 1.x 升级到 2.0.0

1. **备份配置**
   ```bash
   python3 scripts/config_manager.py backup
   ```

2. **更新代码**
   ```bash
   git pull origin main
   ```

3. **更新依赖**
   ```bash
   pip install -r requirements.txt
   ```

4. **迁移配置**
   - 旧配置: `config/config.yaml`
   - 新配置: `config/data_sources.yaml`
   - 环境变量: `.env`

5. **测试系统**
   ```bash
   ./scripts/quick_start.sh check
   ```
