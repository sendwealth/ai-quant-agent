## 💡 盈利策略组合

### 策略1：市场择时 ⭐⭐⭐⭐⭐

**核心思想**: 只在有利市场交易

**实施方法**:
```python
# 1. 判断大盘趋势
if 上证指数 > MA60:
    允许交易
else:
    100%空仓

# 2. 个股筛选
if 价格 > MA30 > MA60:
    加入候选池
else:
    不交易

# 3. 止损策略
if 指数 < MA30:
    全部清仓
```

**预期效果**:
- 避开下跌：减少50%亏损
- 捕获上涨：吃到70%涨幅
- 年化收益：12-18%

---

### 策略2：牛熊分离 ⭐⭐⭐⭐

**核心思想**: 不同市场用不同参数

**牛市参数**:
```python
MA: 10/30 (快速响应)
ATR止损: 3.5倍 (放宽)
仓位: 40% (激进)
追踪止损: 3.0倍
```

**熊市参数**:
```python
MA: 5/20 (快速退出)
ATR止损: 1.5倍 (收紧)
仓位: 10% (保守)
追踪止损: 1.0倍
```

**震荡参数**:
```python
RSI策略: 30买/70卖
仓位: 15%
快进快出
```

**预期效果**:
- 牛市捕获：30-40%收益
- 熊市保护：-5%以内
- 年化收益：15-25%

### 策略5：仓位管理 ⭐⭐⭐⭐

**动态仓位调整**:

```python
# 基于胜率
if 胜率 > 65%:
    基础仓位 = 30%
elif 胜率 > 55%:
    基础仓位 = 20%
else:
    基础仓位 = 10%

# 基于连续盈亏
if 连胜3次:
    仓位 = 基础仓位 * 1.5  # 加仓
elif 连亏2次:
    仓位 = 基础仓位 * 0.5  # 降仓

# 风险控制
if 单日亏损 > 2%:
    停止交易
if 单周亏损 > 5%:
    清仓反思
```

## ⚠️ 风险管理

### 止损规则

| 类型 | 规则 | 执行 |
|------|------|------|
| 个股止损 | ATR × 2.5 | 自动 |
| 日内止损 | -2% | 手动 |
| 周止损 | -5% | 手动 |
| 月止损 | -8% | 手动 |

### 仓位限制

| 情况 | 限制 |
|------|------|
| 单只股票 | ≤40% |
| 同行业 | ≤60% |
| 总股票数 | 3-5只 |
| 现金储备 | ≥20% |

### 风险警示

```
🚨 立即停止交易的情况：
1. 单日亏损 > 2%
2. 连续3次止损
3. 情绪失控（愤怒/恐慌）
4. 偏离交易计划
5. 市场异常波动
```
