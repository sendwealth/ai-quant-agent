## 最佳策略推荐

### 1. 波动率自适应v11 ⭐
**特点**: 根据市场波动率自动调整参数
```python
高波动(>35%): 止损2.0x, 仓位15%
正常波动:     止损2.5x, 仓位25%
低波动(<15%): 止损3.0x, 仓位35%
```
**最佳表现**: 比亚迪 +9.46%

### 2. 保守策略 ⭐
**特点**: 稳定、低风险
```python
MA: 5/30
ATR止损: 2.5倍
仓位: 25%
```
**最佳表现**: 五粮液 +2.82% (vs -16.6%)

## 核心发现

### ✅ 有效场景

| 场景 | 表现 | 推荐策略 |
|------|------|----------|
| 下跌市场 | 减少损失 | 保守策略 |
| 震荡市场 | 小幅盈利 | 混合策略 |
| 高波动上涨 | 捕捉收益 | 波动率自适应 |

### ⚠️ 局限性

| 问题 | 原因 | 影响 |
|------|------|------|
| 上涨跑输大盘 | 止损提前退出 | 错过大涨 |
| 交易频繁 | 信号过多 | 手续费损耗 |
| 样本不足 | 只有3只股票 | 结论有限 |

## 使用建议

### 按市场环境选择

```python
# 判断市场
if price > MA30 > MA60:
    # 上涨趋势
    if volatility < 0.35:
        使用波动率自适应v11 (低波动模式)
    else:
        使用保守策略
        
elif price < MA30 < MA60:
    # 下跌趋势
    使用保守策略 (15%仓位)
    
else:
    # 震荡
    使用混合策略v10
```

### 风险控制

1. **仓位限制**: 单只股票不超过40%
2. **止损纪律**: 严格执行ATR止损
3. **分散投资**: 至少3只股票
4. **回撤控制**: 最大回撤不超过15%
