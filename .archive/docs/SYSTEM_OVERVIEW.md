## 🎯 使用指南

### 快速开始

```bash
# 1. 克隆仓库
git clone https://github.com/sendwealth/ai-quant-agent.git
cd ai-quant-agent

# 2. 安装依赖
pip install pandas numpy akshare

# 3. 运行策略
python3 examples/volatility_adaptive_v11.py

# 4. 运行测试
python3 examples/full_system_test.py
```

### 推荐配置

**保守策略** (下跌市场)
```python
MA: 5/30
ATR止损: 2.5倍
仓位: 25%
```

**波动率自适应** (上涨市场)
```python
低波动: 仓位35%, 止损3.0x
正常: 仓位25%, 止损2.5x
高波动: 仓位15%, 止损2.0x
```
