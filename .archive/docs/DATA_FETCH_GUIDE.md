## 方案B: 手动下载（最稳定）

### 1. 从东方财富下载

**步骤**：
1. 访问：http://quote.eastmoney.com/center/
2. 搜索股票代码（如：600519 茅台）
3. 点击"历史数据"
4. 选择"日K线"，时间范围"3年"
5. 点击"下载" → CSV格式

**保存路径**：`ai-quant-agent/data/real_600519.csv`

### 2. 从网易财经下载

**步骤**：
1. 访问：https://money.163.com/
2. 搜索股票
3. 历史行情 → 导出数据

### 3. 使用通达信/同花顺

**步骤**：
1. 打开软件
2. 查看股票日K线
3. 右键 → 导出数据
4. 选择CSV格式

## 方案C: 使用其他Python库

### TuShare（需要注册）

```python
import tushare as ts

# 设置token（需要注册获取）
ts.set_token('your_token')
pro = ts.pro_api()

# 获取数据
df = pro.daily(
    ts_code='600519.SH',
    start_date='20210301',
    end_date='20260304'
)

# 保存
df.to_csv('data/real_600519.csv', index=False)
```

**优点**：
- 更稳定
- 数据更全
- 速度快

**缺点**：
- 需要注册
- 免费版有积分限制

### yfinance（美股/港股）

```python
import yfinance as yf

# 获取A股（需要加.SS后缀）
stock = yf.Ticker("600519.SS")
df = stock.history(period="3y")

# 保存
df.to_csv('data/real_600519.csv')
```
