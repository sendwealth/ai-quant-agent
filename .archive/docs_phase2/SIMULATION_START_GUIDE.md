## 🎯 第六步：交易规则

### 买入条件（必须同时满足）

1. **MA金叉**
   - 快线上穿慢线
   - 确认趋势向上

2. **MACD确认**
   - MACD柱状图为正
   - 或MACD金叉

3. **RSI适中**
   - 30 < RSI < 70
   - 避免超买超卖

4. **成交量放大**
   - 成交量高于20日均量
   - 确认资金流入

### 卖出条件（满足任一）

1. **止损** ⚠️
   - 价格跌破成本价8%
   - 立即全部卖出

2. **止盈1** 💰
   - 盈利达到10%
   - 卖出50%仓位

3. **止盈2** 💰
   - 盈利达到20%
   - 全部清仓

4. **趋势反转** 🔄
   - MA快线下穿慢线
   - 且MACD柱状图为负
   - 全部卖出

### 仓位管理

**建仓**:
- 初始仓位: 30%
- 分批建仓: 每次30%

**加仓**:
- 盈利5%以上
- 趋势持续向上
- 每次加仓20%

**减仓**:
- 达到止盈1: 减50%
- 达到止盈2: 清仓
- 单只股票最大: 40%
