## 第二步：注册账号 (3分钟)

1. **打开APP**
   - 点击桌面的"同花顺"图标

2. **点击"我的"**
   - 在底部导航栏找到"我的"
   - 点击进入

3. **点击"登录/注册"**
   - 在页面顶部找到
   - 点击进入

4. **输入手机号**
   - 输入你的手机号
   - 点击"获取验证码"

5. **输入验证码**
   - 输入收到的6位验证码
   - 点击"确定"

6. **设置密码**
   - 输入密码（8-20位）
   - 点击"完成注册"

✅ **注册成功！**

4. **点击"立即开户"**
   - 首次使用需要开户

5. **设置初始资金**
   - 输入：**100000** (10万元)
   - 点击"确定"

6. **确认开通**
   - 点击"确认开通"

✅ **模拟账户开通成功！**

## 第六步：设置技术指标 (5分钟)

### 宁德时代和立讯精密（MA10/35）

#### 设置MA指标

1. **进入股票详情页**
   - 在自选页面，点击"宁德时代"

2. **进入K线图**
   - 点击中间区域的K线图
   - 进入全屏K线

3. **添加MA指标**
   - 点击底部的"指标"
   - 找到"MA"（移动平均线）
   - 点击"设置"（齿轮图标）

4. **设置参数**
   - 修改第一条：输入 **10**
   - 修改第二条：输入 **35**
   - 删除第三条、第四条（不需要）
   - 点击"确定"

5. **添加MACD**
   - 点击"指标"
   - 找到"MACD"
   - 点击添加
   - 参数保持默认（12, 26, 9）

6. **添加RSI**
   - 点击"指标"
   - 找到"RSI"
   - 点击添加
   - 参数设置为 **14**

7. **保存设置**
   - 点击右上角"保存"
   - 或直接返回（自动保存）

✅ **宁德时代设置完成！**

#### 重复立讯精密

1. 返回自选页
2. 点击"立讯精密"
3. 重复上述步骤（MA10/35, MACD, RSI）

✅ **立讯精密设置完成！**

## 💡 小贴士

### 如何快速查看MA状态？

1. 打开APP
2. 点击"自选"
3. 点击股票名称
4. 看K线图上的MA线
   - 白色线（快线）在上方 = 多头 ✅
   - 白色线（快线）在下方 = 空头 ❌

### 如何判断金叉？

- 快线（MA10或MA8）从下方上穿慢线（MA35/25/30）
- 形成交叉点
- 这就是金叉 ⚡

### 如何判断死叉？

- 快线从上方下穿慢线
- 形成交叉点
- 这就是死叉 ⚠️
